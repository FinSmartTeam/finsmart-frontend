# FinSmart - Frontend

FinSmart adalah aplikasi pengelolaan keuangan pribadi berbasis web yang memanfaatkan Artificial Intelligence (AI) untuk membantu generasi muda mencatat, menganalisis, dan merencanakan keuangan mereka secara otomatis.

Repositori ini khusus memuat kode sumber untuk sisi **Client/Front-End**.

## Status Pengembangan
Proyek ini sedang dalam tahap awal (Setup Repository & Riset UI/UX). Instruksi instalasi lokal dan panduan menjalankan aplikasi akan diperbarui setelah kerangka awal React.js (Vite) selesai diinisialisasi.

## Rencana Tech Stack
- **Framework:** React.js (Vite)
- **Styling:** Tailwind CSS
- **Visualisasi Data:** Chart.js
- **HTTP Client:** Axios
